# Code of Conduct

## Our Pledge

We pledge to make participation in this project respectful and harassment-free for everyone.

## Standards

Examples of behavior that contributes to a positive environment:

- Using welcoming and inclusive language.
- Respecting differing viewpoints and experiences.
- Giving and gracefully accepting constructive feedback.
- Focusing on what is best for the project and its users.

Examples of unacceptable behavior:

- Harassment, insults, or personal attacks.
- Publishing private information without explicit permission.
- Conduct that would reasonably be considered inappropriate in a professional setting.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, and pull requests that do not align with this code of conduct.
