# Security Policy

## Supported Versions

Security fixes target the default branch.

## Reporting a Vulnerability

Please report vulnerabilities privately through GitHub's private vulnerability reporting flow when available, or contact the repository owner directly.

Do not open public issues for exposed credentials, private invoice data, bank details, tax IDs, or client data.

## Sensitive Data

This project is intended to keep real data local. Never commit:

- API keys or access tokens
- Bank account or routing details
- Tax identifiers
- Real client names, addresses, rates, or invoice data
- Private logos or contract materials
